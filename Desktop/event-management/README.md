# Enterprise Event Management System

A production-ready event management system built with Groovy and Spring Boot.

## Features

- ✅ Event creation & management
- ✅ Ticketing system with QR codes
- ✅ Payment integration (Stripe)
- ✅ Email/SMS notifications
- ✅ Real-time attendee tracking
- ✅ QR code check-in system
- ✅ Admin analytics dashboard
- ✅ WebSocket live updates

## Technology Stack

- **Backend**: Groovy 3.x, Spring Boot 3.1.5
- **Database**: PostgreSQL (with H2 for development)
- **Cache**: Redis
- **Payment**: Stripe API
- **SMS**: Twilio API
- **QR Code**: ZXing
- **Real-time**: WebSockets
